## 说明

椒盐音乐安装包大小约 8 MB ，解压安装后约占用 25 MB 空间（后续使用产生的用户数据不包含）。
**仅提供 64 位版本（架构 arm-v8a），需要安卓 6.0 以上。**

**Google Play 版本由谷歌签名发布，与此仓库安装包不兼容。**
