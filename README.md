建议先查看应用内帮助文档，若无法解决请提交 issue 。注意不接收任何 UI 样式调整相关的反馈（如 “我个人不喜欢某处的界面样子” 等等主观反馈，但不包含 UI 异常错误的反馈），个人独立开发应用很难很难很难做到符合人人的喜欢，就像做菜对吧，咸淡总不同，我通常无法提供多种 “不同口味的菜品” ，希望理解。
